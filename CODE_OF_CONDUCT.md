# 行為準則 (Code of Conduct)

## 我們的承諾

身為 NT-ClawLaunch 專案的成員、貢獻者與維護者，我們承諾為每個人提供一個無騷擾的環境，不論其背景、性別認同、性取向、殘疾、生理外觀、體型、種族、年齡、宗教或技術層次。

我們致力於維護一個友善、安全且具備包容性的社群。

## 我們的標準

為了營造正向環境，我們鼓勵以下行為：
*   展現同理心並善待他人。
*   尊重不同的觀點與經驗。
*   提供建設性的反饋。
*   承擔責任並對受影響的人道歉（如果您做錯了什麼）。
*   專注於對整體社群最有利的事情。

不可接受的行為包括：
*   使用性暗示語言或意象。
*   進行人身攻擊、霸凌、羞辱或貶低他人。
*   在未經許可的情況下公開他人的隱私訊息。
*   騷擾（無論是公開或是私下）。
*   其他可能被合理認為在專業環境中不當的行為。

## 維護者的責任

專案維護者有責任闡明此行為準則的標準，並將針對任何不當、威脅、冒犯或有害的行為採取適當且公平的糾正行動。

## 適用範圍

此行為準則適用於與專案相關的所有實體與虛擬空間，包括 GitHub Issue、Pull Request、討論區以及社交媒體。

## 檢舉流程

如果您遇到或目睹違反此行為準則的行為，請聯繫專案擁有者或維護者（例如透過專案的開發頻道）。所有的舉報都將被保密並得到妥善處理。
