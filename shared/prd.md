# NT-ClawLaunch PRD (需求說明書) - v1.0

## 1. 專案願景
打造一個開源的桌面化 OpenClaw 管理中樞，實現「一鍵安裝」與「可視化監控」，降低用戶進入門檻，並提供具備高級感的 UI 交互體驗。

## 2. 技術選型 (Tech Stack)
- **桌面容器**: Electron
- **前端框架**: React + Vite
- **介面樣式**: Tailwind CSS + shadcn/ui
- **狀態管理**: Zustand
- **通訊協議**: Electron IPC (Inter-Process Communication)

## 3. 核心模組功能需求

### 3.1 導引安裝模組 (Setup Wizard)
- **環境自動偵測**: 
    - 檢查 Node.js (version >= 22.12.0)
    - 檢查 Git 是否安裝
    - 若缺失，顯示導引下載連結或自動化腳本。
- **自動化部署**:
    - 一鍵執行 `git clone` 至用戶指定目錄。
    - 自動執行 `npm/pnpm install` 安裝依賴。

### 3.2 進程監控模組 (Process Manager)
- **核心控制**: 啟動/停止 OpenClaw Gateway。
- **實時日誌流**: 
    - 透過 IPC 監聽後端 `stdout`。
    - 前端 UI 提供自動滾動的「終端機模擬視窗」。
- **健康監測**: 偵測進程崩潰並提供重啟選項。

### 3.3 配置編輯器 (Config GUI)
- **視覺化表單**: 編輯 `openclaw.json` 或 `.env`。
- **秘密管理**: 整合電子加密儲存 (electron-store) 管理 API Keys。
- **模型切換**: 提供下拉選單同步 OpenClaw 支持的模型清單。

### 3.4 統計看板 (Analytics)
- **流量分析**: 解析日誌中的 Token 消耗量。
- **視覺化圖表**: 使用 Recharts 繪製消耗趨勢圖。

## 4. 用戶交互流程 (User Flow)
1. 啟動 App -> 2. 環境檢測 -> 3. 一鍵安裝/路徑配置 -> 4. 儀表板監控 -> 5. 配置調整。
